# @next/ui

Cross-surface UI primitives. Composes [`@next/design-system`](../design-system) tokens.

- `@next/ui/web` — DOM components for Next.js apps (Button, Surface, ThemeProvider).
- `@next/ui/native` — React Native primitives for `apps/mobile`.
- `@next/ui/core` — headless logic shared across both (controlled-state hooks, focus rings, virtualization math).

Surface-specific composition (full screens, layouts, feature components) belongs in the consuming app, not here.
