# @next/api-client

Generated GraphQL + gRPC clients for NEXT.

- GraphQL: codegen via `@graphql-codegen/client-preset`. Run `pnpm codegen` (Turbo-aware) to refresh against the schema URL in `NEXT_GRAPHQL_SCHEMA_URL`.
- gRPC: generated by `buf generate` from the root `buf.gen.yaml`. Stubs land in `src/__generated__/grpc/`.

Generated code is gitignored. CI re-generates on every PR and fails if there's drift between checked-in client usage and the current schemas.
