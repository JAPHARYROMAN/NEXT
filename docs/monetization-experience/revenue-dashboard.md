# Revenue Dashboard

Creator-facing revenue overview in Studio (`/revenue`) built on `@next/revenue-ui`.

## Surfaces

- **Revenue overview** — earnings, members, tips, available payout with sparklines
- **Source breakdown** — subscriptions, sponsorship, commerce, tips, premium
- **Revenue health** — diversification indicator (healthy / stable / attention)
- **Payout status** — available vs pending, scheduled/failed/hold states
- **Ledger preview** — line-item earnings with statement download shell

## Principles

- Transparent splits — no hidden adjustments
- Calm readability — low cognitive load charts
- Filter by range and source without urgency animations

## Integration

Contract-ready UI. Ledger and payout data from commerce/ledger services when available.
