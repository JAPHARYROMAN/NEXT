# Storefront UX

Creator store at `/[handle]/store` via `@next/commerce-ui`.

## Surfaces

- Product grid with digital/bundle/preset cards
- Product detail with calm checkout CTA
- Checkout placeholder (no real payment)
- Purchase history with download/access states

## State

`useStorefrontCartStore` — placeholder cart for demo checkout.

## Telemetry

`trackStorefrontEngagement` for view, add-to-cart, checkout demo.

## Branding

Creator-branded via `ProfileHero` — store feels owned by the creator, not the platform.
