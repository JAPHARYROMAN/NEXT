# Cinematic Environments

Composed environments for watch, discovery, and theater surfaces.

## Layout engine

| Layout                 | Purpose                             |
| ---------------------- | ----------------------------------- |
| `DepthLayout`          | Ambient / content / chrome stacking |
| `CinematicViewport`    | Aspect-aware media viewport         |
| `FocusSensitiveLayout` | Expands focused region              |

## Media shells

- `WatchEnvironment` — playback + atmosphere
- `DiscoveryRoom` — hero + rails with immersive lighting
- `ImmersiveTheaterEnv` — theater-ui + immersive-ui bridge

## Theater / TV

Existing theater and TV flows remain; immersive layers are additive exports, not replacements.
