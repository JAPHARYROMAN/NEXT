# Live Events

## Routes

- Web `/events`, `/events/[id]`
- Studio `/live/events`

## Features

- `LiveCountdown` — premiere badge, tabular timer
- Reminder toggle (placeholder notifications)
- Post-event replay cards + `DiscussionShell`
- TV patterns referenced via shared countdown motion tokens

## State

- `useLiveCountdownStore` — tick + reminder flag
- `trackLiveEventConversion` — view, reminder, join_live

## Placeholders

- Notification delivery, calendar sync
