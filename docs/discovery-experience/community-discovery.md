# Community Discovery

Extended communities page with search, interest clusters, health indicators.

## Route

- `/communities` — enhanced `SocialDiscovery`

## Packages

- `@next/community-ui` — CommunitySearchResults, InterestClusterView, CommunityHealthIndicator
- `@next/social-ui` — CommunityDiscoveryGrid, CulturalWave, EmergingCommunityCard

## Features

- Inline community search with semantic match reasons
- Health score meter (thriving / healthy / emerging)
- Interest cluster navigation
- Existing filter tabs and cultural waves

## Health indicator

Visual meter with `role="meter"` — score 0–1 mapped to label.

## Data

`demoDiscoveryCommunities`, `demoInterestClusters`
