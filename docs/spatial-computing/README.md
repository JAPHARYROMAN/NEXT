# Spatial Computing Bridge

Phase 22 establishes **frontend foundations** for future AR/VR/spatial displays. This folder bridges product docs and implementation.

## Scope (in repo)

- `@next/spatial-ui` — depth navigation
- `@next/immersive-ui` — immersive design language
- `@next/environment-ui` — ambient interfaces
- `@next/ambient-motion` — spatial motion primitives

## Out of scope (Phase 22)

- WebXR / OpenXR runtimes
- Spatial anchors / world mesh
- Backend presence or spatial audio analysis

## Surfaces today

| Surface              | URL / app                       |
| -------------------- | ------------------------------- |
| Web spatial demo     | `http://localhost:3000/spatial` |
| Web ambient demo     | `http://localhost:3000/ambient` |
| Immersive Vite shell | `http://localhost:3011`         |
