# Theater Web Bridge

Theater routes mirror TV routes for projector and ultrawide browsers.

| TV               | Theater alias         |
| ---------------- | --------------------- |
| `/tv`            | `/theater`            |
| `/tv/watch/[id]` | `/theater/watch/[id]` |

Shared implementation: `TheaterHomePage` and `TvWatchPage` under `apps/web/src/features/tv`.

See also: [tv-architecture.md](../tv-experience/tv-architecture.md), [theater-mode.md](../tv-experience/theater-mode.md).
